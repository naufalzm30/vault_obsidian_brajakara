- `06_INDEX/claude_memory/feedback_behavior.md`: mirror sync

**Routing workflow (strict):**
```
Navigation_Map (hop 0) → folder index (hop 1) → detail file (hop 2)
```

**Example:**
- User tanya "identitas" → Navigation_Map → `07_PROFIL/index.md` → `[[identitas]]` (2 hop, konsisten)
- User tanya "tower" → Navigation_Map → `04_INFRASTRUCTURE_REFERENCE/index.md` → `[[Proxmox_MORDOR]]` (2 hop, konsisten)

**Why strict:**
- Konsisten — semua query workflow sama
- Scalable — tambah file baru = update folder index, Navigation_Map tidak touch
- Fast routing — frontmatter metadata (cek YAML tanpa read body)

**Trade-off accepted:**
- Profile/persona queries: 2 hop (Navigation_Map → folder index → detail) vs 1 hop bypass (direct link)
- Tapi frontmatter metadata bikin hop tetap cepat — aku cek `type`, `category`, `hop` tanpa read full content
